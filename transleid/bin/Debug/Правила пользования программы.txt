----------------------------------------------------------------------------------

1) Авторизация - 
Если у вас нету аккаунта то надо нажать registration;
Ввести ник и пароль(будет использоваться для входа) и после нажать registration;
Ввести капчу(защиту от ботов), если капча не понятна, то закройте окно с капчей и нажмите заново registration;
При правильном вводе аккаунт будет зарегистрирован и высветится определенное окно

Если есть аккаунт 
Ввести логин и пароль вводимые в регистрации и нажать Authorization
Remember me - запомнить логин и пароль;

----------------------------------------------------------------------------------

2) Использование переводчика -
1 окно для ввода данных - используется для перевода 1-20 слов
2 окно для файла - используется для перевода 1 файла
3 окно для файла - используется для перевода 2 и более файлов
4 окно для больших текстов(не рекомендуется при переводе более 800 строк)
5 окно для переведенного текста - используется при переводе текста из 1, 2, 3 и 4 окна;

6,7,8 для добавления слов в переводчик

----------------------------------------------------------------------------------

3) Заполнение базы данных переводчика
Есть 2 вида заполнения переводчика
Через файл - Файл находится по пути ../sd/transleit.txt
Заполнять строго - eng:rus:det
Через программу - войти в пользователя - окна 6,7,8 соответственно rus,eng,det


rus - русский язык
eng - английский язык
det - немецкий язык
